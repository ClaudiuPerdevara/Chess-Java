# Chess-Java

♟️ Java MultiChess
A high-performance Java chess application featuring cloud-based multiplayer and a custom AI engine.

🚀 Key Features
Online Multiplayer: Real-time synchronization using Firebase REST API (code-based lobbies).

AI Engine: Local opponent powered by Minimax with Alpha-Beta pruning.

Advanced UI: Java Swing interface with 180° board rotation and real-time evaluation bar.

Full Logic: Supports all FIDE rules (Castling, En Passant, Promotion, Repetition).

🛠️ Tech Stack
Language: Java 25+

Database: Firebase Realtime DB

GUI: Java Swing / AWT
